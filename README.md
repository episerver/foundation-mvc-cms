## Installation

### Windows

```
open command prompt as administrator
git clone git@github.com:episerver/foundation-mvc-cms.git
cd foundation-mvc-cms
git checkout main
setup.cmd 
dotnet run
```

### Mac

```
Open a Terminal window
git clone git@github.com:episerver/foundation-mvc-cms.git
cd foundation-mvc-cms
git checkout main
chmod u+x setup.sh
./setup.sh
dotnet run
```

### Linux

```
Open a bash terminal window
git clone git@github.com:episerver/foundation-mvc-cms.git
cd foundation-mvc-cms
git checkout main
chmod u+x setup.sh
./setup.sh
dotnet run
```
